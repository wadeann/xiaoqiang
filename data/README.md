# 小强量化系统 - A股数据获取模块

## 📋 模块概述

`a_share_data.py` 是小强量化系统的核心数据获取模块，实现了多数据源备份机制，确保数据获取的稳定性和可靠性。

## 🌐 数据源支持

### 1. 新浪财经 (Sina)
- **接口**: `http://hq.sinajs.cn/list={code}`
- **编码**: GBK
- **优点**: 实时性好，免费
- **缺点**: 可能被限流
- **测试成功率**: 100%
- **平均响应时间**: ~1400ms

### 2. 腾讯财经 (Tencent) ⭐ 推荐
- **接口**: `http://qt.gtimg.cn/q={code}`
- **编码**: GBK
- **优点**: 稳定性好，响应快
- **缺点**: 响应稍慢
- **测试成功率**: 100%
- **平均响应时间**: ~1344ms

### 3. 东方财富 (EastMoney)
- **接口**: `http://push2.eastmoney.com/api/qt/stock/get`
- **市场代码**: 1=沪市, 0=深市
- **优点**: 数据全面
- **缺点**: 超时较多
- **测试成功率**: 33% (指数获取不稳定)
- **平均响应时间**: ~1800ms

## 🚀 使用方法

### 基本用法

```python
from a_share_data import AShareDataSource

# 创建数据源实例
data_source = AShareDataSource(timeout=10, retry=3)

# 获取单个股票行情
quote = data_source.get_quote('sh600519')
print(f"贵州茅台: {quote['price']}")

# 获取主要指数行情
indices = data_source.get_index_quotes()

# 获取热门股票行情
hot_stocks = data_source.get_hot_stocks_quotes()

# 获取AI/半导体板块行情
ai_stocks = data_source.get_ai_stocks_quotes()

# 打印市场概况
data_source.print_market_summary()
```

### 股票代码格式

- 沪市股票: `sh` + 代码 (如 `sh600519`)
- 深市股票: `sz` + 代码 (如 `sz000858`)
- 沪市指数: `sh` + 代码 (如 `sh000001`)
- 深市指数: `sz` + 代码 (如 `sz399001`)

## 🔄 自动切换机制

模块实现了自动数据源切换功能：

1. 按优先级尝试数据源: sina → tencent → eastmoney
2. 当某个数据源失败或返回无效数据时，自动切换到下一个
3. 每个数据源支持重试机制（默认3次）
4. 批量获取时自动添加延迟，避免频率限制

## 四、选股方法论

### 买入信号 (5个条件)

```python
1. 60天涨幅 < 10%     # 低位
2. RSI 30-50          # 有上涨空间
3. 价格分位 < 30%     # 历史低位
4. 成交量放大 > 1.5x  # 资金关注
5. 刚突破 MA20        # 趋势启动
```

### 不买信号 (5个条件)

```python
1. 60天涨幅 > 50%     # 涨幅过大
2. RSI > 70           # 超买
3. 价格分位 > 80%     # 高位
4. 成交量萎缩 < 0.8x  # 资金流出
5. 跌破 MA20          # 趋势破坏
```

### 止损止盈策略

```python
止损: -6% (严格执行)
止盈: +30% 到 +50% (分批止盈)
移动止损: 从最高点回撤 8%

仓位控制:
  单只股票 ≤ 20%
  单板块 ≤ 40%
  现金 ≥ 20%
```

price_percentile = (df['close'] < current_price).sum() / len(df) * 100

# RSI
delta = df['close'].diff()
gain = (delta.where(delta > 0, 0)).rolling(14).mean()
loss = (-delta.where(delta < 0, 0)).rolling(14).mean()
rsi = 100 - (100 / (1 + gain / loss))

# 成交量放大
